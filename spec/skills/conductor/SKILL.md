## PURPOSE
One command -> a complete, ready-to-use account package: research, strategy, and all-tier outbound copy, assembled into a single artifact Justin can act on (and drop verified contacts into his prebuilt sequences).

## TRIGGERS
"conductor", "blaze", "run conductor/blaze on", "prep me for X", "get me ready for X", a brand/URL drop where Justin wants the full package or fast call-prep, and closed-lost re-engagements.

## CONDUCTOR DOES (uses ALL skills every time; gift only when asked)
1. RECON -> intel packet + verified, tiered roster + prior context + guardrail result + Gift Target. If a guardrail fails -> drop & replace (or STOP and explain for a single-account request).
2. KYLE -> fit, commercial POV, Solution Hypothesis + Business Case, Target Champion, objection prep, call + LinkedIn strategy.
3. VINCI -> A/B emails for EVERY tier (T1/T2/T3), 6-8 emailed per company (list extras beyond 8); T3 also keeps the full call list.
4. SANTA -> ONLY if Justin asks for a gift.
5. ASSEMBLE one package (below). Closed-lost re-engagement: reference the prior deal + lost reason and lead with what changed.

## CONDUCTOR DOES NOT
Re-implement research, strategy, or copywriting itself, and does NOT build multi-step send sequences (Justin enrolls into prebuilt sequences). It calls Recon/Kyle/Vinci/(Santa) and stitches the result. No duplicated logic, no contradictions - each piece comes from its owner skill.

## OUTPUT (copy box - single package)
- TOP BANNER: guardrail/parent result; warm vs fresh; any follow-up ask (who/when/context); closed-lost note if applicable.
- COMPANY OVERVIEW (~500 words): what they do, 2025 unit count + band, leadership/funding/news, the Solution Hypothesis + Business Case, and the named Target Champion.
- CONTACT LIST by tier (T1 -> T2 -> T3): every contact = name, role, LinkedIn (hyperlinked), all phones (with HQ/shared flags), Denver tag, email (or flagged pattern guess), HubSpot status. Each contact (T1, T2 AND T3) carries TWO distinct A/B intro emails from Vinci; T3 ALSO shows the call list. Cap emails at 6-8 contacts; list extras without emails.
- OPTIONAL: gift brief (only if asked).
- One artifact, ready to use. DRAFTS ONLY.

## DONE CHECK
All skills invoked (Santa only if asked); guardrails clear/flagged; overview has hypothesis + business case + champion; every shown contact verified with A/B emails + call data; nothing duplicated across sections.
